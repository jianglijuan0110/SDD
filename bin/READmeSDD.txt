READ ME :


Ce code a pour but de creer des sacs de de classes genérique, ces classes doivent implémenter l'interface "PoidsAndType".
Donc on peut creer d'autres classes qui peuvent être mis dans la classe sac sous conditions que ces classes implémente l'interface.
Ici j'ai creer une classe "potato" afin de creer des sac de patates, vous pouvez creer des classes fruits, ou légumes ou autres..

La classe Sac met en valeur les arraylist et leur différentes methodes :

	.size() pour avoir la taille de notre arrayList
	.contains() pour savoir si un element existe dans notre arrayList
	.isEmpty() pour voir si notre arraylist est null ou non
	.add() pour rajouter un élement à notre arrayList
	.remove() pour supprimer un élement de notre arrayList
	.removeAll() pour supprimer le contenu d'une arrayList B d'une arrayList A

De plus une itération classique sur les élement de l'arrayList (qui met en valeur son coté iterable) à été utilisé

le main explique toute les méthode utilisées, avec un chrono sur le déroulement de ce main vers la fin.
